# silo-report

Generate GP and VWAP breakdown by silo (retail/wholesale/custody) and channel (digital/dealer).

## When to use
Weekly or monthly, or whenever leadership needs a breakdown of where GP is coming from by silo or channel.

## Pull silo analytics for a date range

```python
from models import get_deals
from processor import calc_silo_analytics, calc_channel_analytics
import json

entity     = 'SABIS'
metal      = 'gold'
# For a specific date range, pull all deals and filter
deals      = get_deals(entity, metal)  # returns all deals

# Filter to desired date range (optional)
from_date = '2026-01-01'
to_date   = '2026-01-31'
deals_filtered = [d for d in deals if from_date <= d['deal_date'] <= to_date]

silo_data    = calc_silo_analytics(deals_filtered)
channel_data = calc_channel_analytics(deals_filtered)

print("=== SILO BREAKDOWN ===")
for silo, s in silo_data.items():
    print(f"{silo.upper()}: {s['deal_count']} deals | {s['total_oz']:.4f}oz | "
          f"GP: R{s['gp_contribution_zar']:,.0f} ({s['gp_proportion_pct']}%) | "
          f"Buy VWAP margin: {s['buy_vwap_margin']:+.2f}% | Sell: {s['sell_vwap_margin']:+.2f}%")

print("\n=== CHANNEL BREAKDOWN ===")
for ch, c in channel_data.items():
    print(f"{ch.upper()}: {c['deal_count']} deals | {c['total_oz']:.4f}oz | "
          f"GP: R{c['gp_contribution_zar']:,.0f} ({c['gp_proportion_pct']}%)")
```

## Dashboard view
The silo and channel breakdown charts are live on the dashboard at [http://localhost:5000](http://localhost:5000) — they update every 30 seconds and reflect today's deals automatically.

## API endpoints

| Endpoint | Purpose |
|---|---|
| `GET /api/analytics/silo?entity=SABIS&metal=gold&date=2026-01-15` | Silo breakdown for a specific date |
| `GET /api/analytics/channel?entity=SABIS&metal=gold` | Channel breakdown (all time) |

## Key metrics explained

| Metric | Meaning |
|---|---|
| `gp_contribution_zar` | Total ZAR gross profit generated by this silo/channel |
| `gp_proportion_pct` | % share of total day's GP this silo/channel accounts for |
| `buy_vwap_margin` | Weighted average buy-back margin (% under spot, weighted by oz) |
| `sell_vwap_margin` | Weighted average sell margin (% over spot, weighted by oz) |
| `vwap` | Overall deal VWAP (ZAR per oz) for this silo/channel |
