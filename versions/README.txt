Treasury Brain — Version Snapshots
===================================

Each folder is a complete snapshot of the app at a specific version.

To RESTORE a version:
  1. Open the version folder in File Explorer
  2. Select all (Ctrl+A) and Copy (Ctrl+C)
  3. Navigate to the project root and Paste (Ctrl+V)
  4. Confirm "Replace" when prompted
  5. Run start.bat

Versions are also tagged in git and visible on GitHub:
  https://github.com/suhaylobrien-blip/Treasury-Brain/tags
